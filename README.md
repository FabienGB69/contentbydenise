# Denise Fabiano — Marketing Website (V1)

Premium, conversion-focused Next.js marketing website for **Denise Fabiano**, a Belgium-based multilingual UGC & Event Content Creator.

## Stack
- Next.js (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion (light use in hero animation)

## Setup
1. Install dependencies:
   ```bash
   npm install
   ```
2. Run development server:
   ```bash
   npm run dev
   ```
3. Build for production:
   ```bash
   npm run build
   ```
4. Start production server:
   ```bash
   npm run start
   ```
